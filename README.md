# Rattrapage – Qualité & Tests

Ce sujet de rattrappage présente une analyse de code, une stratégie de tests,
l’implémentation de tests automatisés ainsi qu’une analyse de métriques
de qualité logicielle.


## Exécution des tests

Les tests automatisés sont implémentés avec le framework Jest.

### Prérequis
- Node.js installé (version 16 ou supérieure recommandée)
- Dépendances installées via npm

### Installation des dépendances

```bash
npm install
npm test
